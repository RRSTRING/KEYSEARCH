Game Description

Exciting game features and fun gameplay! KEYSEARCH is a game where once you click on any letter on your keyboard, it transports you into a random minigame. Inspired by Google, KEYSEARCH is a game made for fun. If anyone expects high-quality gaming experience, a note for you, the game doesn't have great graphics as it's supposed to be a '90s browser. Also, watch out for LOOOOOOOOOOORRRREEEE.

Features

  • Multiple hidden secrets
